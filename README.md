# 📡 Telink BLE Mesh Hardware Control Core (C++)

> **본 저장소는 텔링크(Telink) 동글 하드웨어의 순정 프로토콜 및 BLE 메쉬(Mesh) 통신 선로를 로우 레벨에서 정밀 제어하는 C++ 핵심 소스코드 엔진입니다.**
> 추후 C# 환경으로의 무결점 포팅 및 아키텍처 변환 분석을 위한 원천 도면으로 활용됩니다.

---

## 🛠️ 핵심 분석 및 포팅 목표 (C++ to C#)
* **SNO 카운터 싱크 무결성:** 동글 내부 레지스터와 상위 엔진 간의 하드웨어 일련번호(`SnoCounter`) 리틀 엔디안 조립 구조 분석
* **IV Index 세션 추종:** 무선 채널 개방 시 실시간 주파수 비콘 및 보안 세션 엇박자 원천 차단 로직 이식
* **AppKeyIndex 동적 매핑:** 가짜 고정 상수를 숙청하고 JSON 도면에서 암호화 키 일련번호를 실시간 적출하는 선로 변환
* **하드웨어 뇌사(Freeze) 방지:** 플래시 메모리 라이팅(`0xE9 0xFF 0x03`) 및 선제 리셋 직후의 물리적 딜레이 마진 최적화
* **특정기능 우선사용:** 장치초기화(GwReset), Json파일가져오기(InputDB), 조명제어(Mesh)기능

* `c_shap/` : 텔링크 c++프로그램의 특정기능을 가져와서 작업한 소스

## 📡 HCI 순정 게이트웨이 명령셋 레퍼런스
본 코드가 다루는 핵심 하드웨어 패킷 규격입니다.

| 명령 코드 (HEX) | 기능 명칭 | 물리적 하드웨어 동작 |
| :--- | :--- | :--- |
| `E9 FF 02` | HCI_RESET | 동글 드라이버 완전 세탁 및 포트 재부팅 |
| `E9 FF 03` | HCI_SAVE_START | 개별 조명 디바이스 키 플래시 영구 라이팅 마감 |
| `E9 FF 0F` | HCI_SET_SNO | 동글 심장에 진짜 SNO 카운터 및 IV 주파수 동화 주입 |
| `E9 FF 12 01` | HCI_ANT_OPEN | 무선 안테나 파이프라인 및 실시간 스캔 통로 개방 |
